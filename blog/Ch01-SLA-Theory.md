# 임용 영교론 Ch.1 SLA Theory 완전 정리: Krashen 5대 가설부터 Sociocultural까지

영교론 13개 챕터 중 **SLA Theory가 압도적 최빈출**이다. <span style="color:#3182ce;">**Input · Interaction · Output**</span> 3대 가설은 거의 매년 출제되고, <span style="color:#3182ce;">**Sociocultural Theory**</span>의 ZPD·Scaffolding까지 합치면 한 해 한 문항은 거의 확정.

이 글은 서브노트에 정리된 9개 이론을 더 풀어서 정리한다. **① 개념 정의**(영어 원문 + 한글 풀이) → **② 기출 맥락**(연도·답안 형식) → **③ 헷갈리는 key terms** → **④ 합격자 노트 답안 작성 팁** 4섹션. 끝에 **개념 정리 페이지**와 **OX 20문항**도 임베드해뒀다.

---

## ① 개념 정의 — 9개 이론을 영어 원문과 함께

### 1) Behaviorism

> Behaviorism views language acquisition as a process of habit formation through stimulus-response conditioning. Learners imitate input, receive feedback, and gradually establish correct habits via reinforcement.

<span style="color:#805ad5;">**Skinner**</span>의 <span style="color:#3182ce;">**Behaviorism**</span>은 language acquisition을 stimulus-response conditioning에 의한 habit formation으로 본다. learner는 input을 모방하고, feedback을 받으며, reinforcement를 통해 올바른 habit을 점진적으로 형성한다. <span style="color:#805ad5;">**Chomsky**</span>가 *Verbal Behavior* 서평에서 "creativity of language"를 들어 강하게 반박하면서 자리를 내준다.

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 답안에 "Behaviorism은 <span style="color:#3182ce;">**Audiolingual Method**</span>의 이론적 기반"이라고 한 문장 박아주면 채점관이 맥락 잡기 좋다.

### 2) Universal Grammar (UG)

> Universal Grammar holds that humans are born with an innate language faculty containing principles common to all languages, allowing children to acquire language despite limited and imperfect input — known as the Poverty of the Stimulus.

<span style="color:#805ad5;">**Chomsky**</span>의 <span style="color:#3182ce;">**Universal Grammar**</span>는 인간이 모든 언어에 공통된 principles를 담은 innate language faculty를 갖고 태어난다고 본다. 아이들은 제한적이고 불완전한 input만으로도 언어를 acquire하는데, 이를 <span style="color:#3182ce;">**Poverty of the Stimulus**</span>라고 한다. SLA에서는 "L2 learner도 UG에 access하느냐"가 쟁점.

### 3) Krashen's 5 Hypotheses ⭐

<span style="color:#805ad5;">**Krashen**</span>은 다섯 개의 가설을 묶어 <span style="color:#3182ce;">**Monitor Model**</span>로 제안했다. 영교론에서 가장 자주 나오는 핵심.

#### 3-1. Acquisition-Learning Hypothesis

> Acquisition is a subconscious process similar to first language development, while learning refers to conscious knowledge of grammar rules. The two are entirely separate; learning cannot turn into acquisition.

<span style="color:#3182ce;">**Acquisition**</span>은 모국어 습득과 유사한 <span style="color:#319795;">subconscious</span> 과정이고, <span style="color:#3182ce;">**Learning**</span>은 grammar rules에 대한 <span style="color:#c53030;">conscious</span> 지식이다. 두 process가 **완전히 분리**되어 있고 learning이 acquisition으로 변환되지 않는다는 게 <span style="color:#c53030;">**Non-interface position**</span>. 반대 입장이 <span style="color:#805ad5;">**Bialystok**</span>의 <span style="color:#c53030;">**Interface position**</span> — ③ 키텀 비교에서 정리.

#### 3-2. Monitor Hypothesis

> Conscious learning serves only as a Monitor that checks output for accuracy after acquisition has produced the utterance. The Monitor operates only when the speaker has time, focuses on form, and knows the rule.

learning을 통해 얻은 conscious 지식은 acquisition이 생성한 발화를 사후에 점검하는 <span style="color:#3182ce;">**Monitor**</span> 역할만 한다. Monitor 작동 조건 세 가지: ① 시간이 있을 것 ② form에 집중할 것 ③ 그 rule을 알고 있을 것. 시험·작문에는 도움이 되지만 자연스러운 대화에서는 거의 못 쓴다.

#### 3-3. Natural Order Hypothesis

> Grammatical structures are acquired in a predictable order regardless of the learner's L1, age, or instruction.

grammatical structures는 L1·나이·instruction과 무관하게 예측 가능한 <span style="color:#3182ce;">**Natural Order**</span>로 acquire된다. 영교론에서는 단독 출제가 드물지만 <span style="color:#805ad5;">**Pienemann**</span>의 <span style="color:#3182ce;">**Processability Hypothesis**</span>와 묶어서 출제된다.

#### 3-4. Input Hypothesis (i+1) ⭐

> Comprehensible input slightly beyond the learner's current level (i+1) is the necessary and sufficient condition for SLA. Acquisition takes place when learners understand input that contains structures slightly above their current competence.

learner의 현재 수준(i)보다 약간 높은 <span style="color:#3182ce;">**comprehensible input (i+1)**</span>이 SLA의 필요충분 조건이다. learner가 자신의 current competence보다 약간 위의 구조를 포함한 input을 이해할 때 acquisition이 일어난다. (<span style="color:#c53030;">**2011 기출**</span>)

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 답안에는 `comprehensible input (i+1)`을 영어 그대로. <span style="color:#c53030;">**"이해 가능한 입력"으로 풀어 쓰면 감점**</span>.

#### 3-5. Affective Filter Hypothesis

> Affective variables — motivation, self-confidence, and anxiety — act as a filter that determines how much input reaches the language acquisition device. A high filter blocks input; a low filter lets it through.

motivation·self-confidence·anxiety 같은 정서적 변인이 <span style="color:#3182ce;">**Affective Filter**</span>를 형성해 얼마나 많은 input이 language acquisition device에 도달할지 결정한다. high filter면 input이 차단되고, low filter면 통과된다. 같은 i+1이 주어져도 학생이 anxious하면 acquisition이 일어나지 않는다.

### 4) Output Hypothesis (Swain) ⭐

> Comprehensible output pushes learners to process language syntactically rather than just semantically. The functions of output include noticing the gap, hypothesis testing, and metalinguistic reflection.

<span style="color:#805ad5;">**Swain**</span>의 <span style="color:#3182ce;">**Output Hypothesis**</span>는 comprehensible output이 learner로 하여금 의미(semantically)뿐 아니라 통사적(syntactically)으로도 언어를 처리하도록 밀어붙인다고 본다. output의 세 가지 functions:

1. <span style="color:#3182ce;">**Noticing the gap**</span> — 자신이 말하고 싶은 것과 실제 말한 것 사이의 차이 인식
2. <span style="color:#3182ce;">**Hypothesis testing**</span> — 가설을 시험하는 도구
3. <span style="color:#3182ce;">**Metalinguistic reflection**</span> — 언어에 대한 반성적 사고

(<span style="color:#c53030;">**2011·2012 기출**</span>)

### 5) Noticing Hypothesis (Schmidt) + HIG·LIG

> Schmidt's Noticing Hypothesis claims that conscious attention to formal features in input is a necessary condition for converting input into intake.

<span style="color:#805ad5;">**Schmidt**</span>의 <span style="color:#3182ce;">**Noticing Hypothesis**</span>는 input의 formal features에 대한 conscious attention이 input을 <span style="color:#3182ce;">**intake**</span>로 변환시키는 necessary condition이라고 본다. <span style="color:#805ad5;">Krashen</span>의 unconscious model과 정면으로 충돌.

연관 개념: <span style="color:#3182ce;">**HIG (High Input Generator)**</span> vs <span style="color:#3182ce;">**LIG (Low Input Generator)**</span> — 같은 환경에서도 input을 적극적으로 끌어내는 learner가 더 잘 acquire한다.

### 6) Information Processing

> Information processing models distinguish controlled processing (slow, conscious, attention-demanding) from automatic processing (fast, subconscious, effortless). Practice transforms controlled into automatic.

<span style="color:#3182ce;">**Controlled processing**</span>(느리고 의식적, attention 필요) vs <span style="color:#3182ce;">**Automatic processing**</span>(빠르고 자동적). practice가 controlled를 automatic으로 전환시킨다. 이게 바로 <span style="color:#805ad5;">Bialystok</span>의 <span style="color:#c53030;">**Interface position**</span>이 기대는 메커니즘.

### 7) Processability Hypothesis (Pienemann)

> Pienemann's Processability Hypothesis states that learners can only acquire features they are developmentally ready to process. The Teachability Hypothesis follows: instruction is effective only when it targets the next stage in the developmental sequence.

<span style="color:#805ad5;">**Pienemann**</span>의 <span style="color:#3182ce;">**Processability Hypothesis**</span>는 learner가 developmentally 처리할 준비가 된 features만 acquire할 수 있다고 본다. 그래서 <span style="color:#3182ce;">**Teachability Hypothesis**</span>가 도출됨 — 수업은 learner의 발달 단계 바로 다음 단계를 가르칠 때만 효과적.

### 8) Interaction Hypothesis (Long) ⭐⭐

> Long's original Interaction Hypothesis (1981, 1983) proposes that conversational interaction makes input comprehensible through negotiation of meaning. The updated version (1996) emphasizes that interaction promotes acquisition by providing opportunities for learners to receive feedback, notice gaps, and produce modified output.

<span style="color:#805ad5;">**Long**</span>의 <span style="color:#3182ce;">**Interaction Hypothesis**</span>는 두 버전이 있고, 둘을 구분하는 게 시험에서 핵심이다.

| 버전 | 연도 | 초점 |
|---|---|---|
| **Original** | 1981, 1983 | <span style="color:#319795;">**input을 comprehensible하게**</span> — interaction이 input을 이해 가능하게 만든다 |
| **Updated** | 1996 | <span style="color:#319795;">**output을 comprehensible하게**</span> — interaction이 learner에게 feedback과 modified output 기회 제공 |

(<span style="color:#c53030;">**2009·2009모의·2020 기출**</span>)

interaction의 핵심 도구는 <span style="color:#3182ce;">**Negotiation of meaning**</span>이며, 가장 효과적인 전략은 <span style="color:#3182ce;">**Clarification request**</span>로 본다. (<span style="color:#c53030;">**2009·2012·2020 기출**</span>)



### 9) Sociocultural Theory (Vygotsky)

> Vygotsky's Sociocultural Theory views learning as inherently social, mediated by interaction with more knowledgeable others. Two key concepts: the Zone of Proximal Development (ZPD) — the gap between what a learner can do alone and with help — and Scaffolding — the support that bridges this gap.

<span style="color:#805ad5;">**Vygotsky**</span>의 <span style="color:#3182ce;">**Sociocultural Theory**</span>는 learning을 본질적으로 social한 것으로 보며, <span style="color:#3182ce;">**MKO (More Knowledgeable Other)**</span>와의 interaction을 통해 mediate된다. 핵심 두 개념:

- <span style="color:#3182ce;">**ZPD (Zone of Proximal Development)**</span>: 혼자서 할 수 있는 것과 도움받아 할 수 있는 것 사이의 영역
- <span style="color:#3182ce;">**Scaffolding**</span>: 그 영역을 메우는 support

(<span style="color:#c53030;">**2018 기출**</span>)

## ② 기출 맥락 — 어떻게 출제됐나

영교론 SLA Theory는 답안 형식이 거의 정해져 있다. 합격자 노트와 기출분석을 종합한 빈출 패턴.

### 자주 출제된 포인트

- <span style="color:#c53030;">**2011 기출**</span>: <span style="color:#805ad5;">Krashen</span>의 <span style="color:#3182ce;">**Input Hypothesis**</span>와 i+1 정의 (+ <span style="color:#3182ce;">**Acquisition-Learning**</span>·<span style="color:#3182ce;">**Affective Filter**</span> 묶음 분석)
- <span style="color:#c53030;">**2011·2012 기출**</span>: <span style="color:#805ad5;">Swain</span>의 <span style="color:#3182ce;">**Output Hypothesis**</span> + <span style="color:#3182ce;">**Pushed output**</span>
- <span style="color:#c53030;">**2009·2009모의·2020 기출**</span>: <span style="color:#805ad5;">Long</span>의 <span style="color:#3182ce;">**Interaction Hypothesis**</span> 및 <span style="color:#3182ce;">**Negotiation of meaning**</span>
- <span style="color:#c53030;">**2018 기출**</span>: <span style="color:#3182ce;">**Sociocultural perspective**</span> (ZPD·Scaffolding·Regulation 묶음 기입형)
- <span style="color:#c53030;">**2018 기출**</span>: <span style="color:#3182ce;">**Cognitive perspective**</span> (Information Processing·Declarative→Procedural·Automatization)

### 답안 작성 형식

영교론 SLA 답안은 거의 다음 패턴 중 하나로 떨어진다:

```
[Item A] does/shows X in that [evidence].
[Item B] does/shows Y since [evidence].
```

또는

```
The teacher's behavior reflects [theory] by [specific action]
because [theoretical mechanism].
```

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: <span style="color:#c53030;">**"In this passage..." 도입은 절대 금지**</span>. 첫 문장은 바로 The teacher / The activity / Sentence (X) is ... 로 들어가는 게 합격자 패턴.

## ③ 헷갈리는 key terms 비교

### Acquisition vs Learning (Krashen)

| | Acquisition | Learning |
|---|---|---|
| 의식 | <span style="color:#319795;">subconscious</span> | <span style="color:#c53030;">conscious</span> |
| 비유 | 모국어 습득 | grammar 학습 |
| 결과 | implicit knowledge | explicit knowledge |
| 변환 가능성 | — | <span style="color:#c53030;">acquisition으로 변환 X (Non-interface)</span> |

### Non-interface vs Interface

| | Non-interface (<span style="color:#805ad5;">Krashen</span>) | Interface (<span style="color:#805ad5;">Bialystok</span>) |
|---|---|---|
| explicit → implicit | <span style="color:#c53030;">불가능</span> | <span style="color:#319795;">가능 (practice 통해)</span> |
| 함의 | grammar 수업 무용 | grammar 수업 유용 |
| 메커니즘 | acquisition 단일 경로 | controlled → automatic |

### Original vs Updated Interaction Hypothesis

| | Original (1981, 1983) | Updated (1996) |
|---|---|---|
| 초점 | <span style="color:#319795;">**comprehensible input**</span> 만들기 | <span style="color:#319795;">**comprehensible output**</span> 끌어내기 |
| 영향 | <span style="color:#805ad5;">Krashen</span> 연장선 | <span style="color:#805ad5;">Swain</span> 흡수 + feedback 강조 |
| 핵심 메커니즘 | negotiation of meaning | feedback, noticing the gap, modified output |

### Teachability vs Krashen's Zero Option

| | Teachability (<span style="color:#805ad5;">Pienemann</span>) | Zero Option (<span style="color:#805ad5;">Krashen</span>) |
|---|---|---|
| 명시적 instruction | <span style="color:#319795;">발달 단계 다음 단계라면 효과적</span> | <span style="color:#c53030;">아예 무용</span> |
| 가르치는 방식 | 발달 순서대로 | comprehensible input 노출만 |

### Scaffolding 두 정의

수험생이 가장 많이 헷갈리는 부분.

- **정의 ①**: <span style="color:#319795;">supportive environment</span> — learner가 안전하게 시도해볼 수 있는 환경 자체
- **정의 ②**: <span style="color:#319795;">interactional supports</span> — 교사·동료가 제공하는 구체적 발화 지원 (questioning, prompting, modeling)

(<span style="color:#c53030;">**2018 기출**</span>)

## ④ 합격자 노트 기반 답안 작성 팁

### 함정 1: <span style="color:#805ad5;">Krashen</span>의 acquisition을 일반 language acquisition과 혼동

답안에 "language acquisition is ..."로 시작하면 채점관이 어떤 정의인지 모른다. <span style="color:#c53030;">**Krashen의 acquisition은 항상 "subconscious acquisition"으로 명시**</span>해야 한다.

### 함정 2: i+1을 한국어로 풀어 쓰기

<span style="color:#c53030;">**"이해 가능한 입력 +1"이라고 풀어 쓰지 말 것**</span>. 합격자 답안은 모두 `comprehensible input (i+1)` 영문 그대로 쓴다.

### 함정 3: <span style="color:#805ad5;">Swain</span>의 output 세 functions를 두 개만 쓰기

세 개를 다 쓰자: <span style="color:#319795;">**noticing the gap, hypothesis testing, metalinguistic reflection**</span>. 두 개만 쓰면 부분점수 손해. 시간이 빠듯하면 한 줄에 묶어서라도 세 개 모두 언급.

### 함정 4: <span style="color:#3182ce;">**Interaction Hypothesis**</span>를 한 버전만 알고 들어가기

<span style="color:#c53030;">**Original vs Updated 구분이 답안의 절반**</span>. 발문에서 어느 쪽을 묻는지 잡고, 짧게라도 다른 버전과의 차이를 한 줄 추가하면 만점에 가까워진다.

### 함정 5: <span style="color:#3182ce;">**Scaffolding**</span>을 ZPD와 동일시

ZPD는 영역(zone), Scaffolding은 그 영역을 메우는 support. **다른 개념**이다. 답안에는 두 정의를 분리해서 쓰자.

### 답안 시작 패턴 (반드시 외울 것)

- ❌ "In this passage, the teacher uses..."
- ❌ "It can be seen that..."
- ✅ "The teacher's behavior reflects [theory] in that ..."
- ✅ "Activity (A) shows [feature] since ..."
- ✅ "[Item] applies/shows [X] by [-ing form]."

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 영교론 답안은 <span style="color:#c53030;">**한 채점 기준 항목당 1~2문장, 총 3~5문장**</span>이 모범. 강사처럼 길게 쓰면 채점 포인트가 흐려져서 오히려 손해.

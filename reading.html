기출 2009모의.23 Bottom-up / Top-down
Bottom-up processing begins with the reader's background knowledge and expectations, using prior experience to predict text meaning before decoding individual words.
🔧 핵심 정정
위 설명은 top-down processing의 정의입니다. Bottom-up processing은 가장 작은 언어 단위(글자 → 단어 → 문장 → 텍스트)를 해독하는 데서 출발하는 처리 방식입니다.
📖 상세 설명
Bottom-up processing: 가장 작은 언어 단위(글자 → 단어 → 문장 → 전체 텍스트)를 순차적으로 해독하여 의미를 구성하는 data-driven, text-based 방식. 독자의 배경지식보다는 텍스트 자체에 집중하며, phonics approach나 intensive reading과 연결됩니다.

Top-down processing은 반대로 독자의 schematic knowledge(배경지식)를 활용해 텍스트를 예측하고 의미를 재구성하는 conceptually-driven, knowledge-based 방식으로 whole language approach·extensive reading과 연결됩니다.
Bottom-up processing: Decoding process in which meaning is decoded and built up from the smallest units of the given input to a complete text. (lower-level skill; data-driven; text-based; focuses on lg itself)
📊 비교 정리 — Bottom-up vs Top-down
Bottom-up
글자 → 단어 → 문장 → 텍스트 / data-driven / Phonics / Intensive reading
Top-down
배경지식 → 예측 → 확인 / conceptually-driven / Whole Language / Extensive reading
2
기출 2009모의.23 Content / Formal Schema
Content schemata include a reader's background knowledge about organizational forms, discourse structures, and rhetorical patterns of different text types.
🔧 핵심 정정
위 설명은 formal schemata의 정의입니다. Content schemata는 일반적 세계 지식, 사회문화적 지식, 주제 지식(general world / sociocultural / topic knowledge)을 포함합니다.
📖 상세 설명
Content schemata: general world knowledge + sociocultural knowledge + topic knowledge. 독자가 텍스트 내용에 대해 갖는 배경지식의 총체로, 같은 글을 읽어도 전공이 다른 독자가 다르게 추론하는 것이 content schemata의 차이 때문입니다 (기출 2009모의.23 실험).

Formal schemata: background knowledge about organizational forms, discourse structures and rhetorical structures of different texts (e.g. essay 구조, 비교·대조 문단 패턴, genre knowledge).
Content Schema: general world knowledge, sociocultural knowledge, topic knowledge. / Formal schema: background knowledge about organizational forms and discourse, rhetorical structures.
📊 비교 정리 — Content vs Formal Schemata
Content schemata
일반 세계지식 / 사회문화지식 / 주제지식 → 내용에 대한 배경지식
Formal schemata
담화구조 / 수사적 패턴 / 텍스트 유형 지식 → 글의 형식에 대한 배경지식
3
기출 2015.A.기입5 Extensive Reading
Extensive reading involves reading short passages for the explicit purpose of learning new reading skills and language items, usually followed by comprehension exercises.
🔧 핵심 정정
위 설명은 intensive reading의 정의입니다. Extensive reading은 즐거움과 실생활 목적을 위해 학습자가 스스로 선택한 긴 텍스트를 읽는 것으로, 교실 comprehension 연습에 집중하지 않습니다.
📖 상세 설명
Extensive reading: Reading longer texts which are selected by Ss themselves, for pleasure and for real-world purposes, without a focus on classroom exercises designed to test comprehension skills. → top-down approach 기반, implicit learning 및 bottom-up 읽기 기능의 자동화(automatization) 유도. book-conferencing·sustained silent reading 포함.

Intensive reading: 짧은 텍스트를 깊이 이해하며 reading skill·vocabulary 등을 명시적으로 학습. comprehension exercise가 뒤따름 → bottom-up approach 기반.
Extensive reading: Reading longer texts which are selected by Ss themselves, for pleasure and for real-world purposes, without a focus on classroom exercises designed to test comprehension skills.
📊 비교 정리 — Intensive vs Extensive Reading
Intensive reading
짧은 텍스트 / 기능·어휘 명시적 학습 / comprehension 연습 포함 / bottom-up
Extensive reading
긴 텍스트 / 즐거움·실생활 목적 / 학생 자기선택 / top-down / 묵독
4
기출 2011.17 · 2013.22 Reconstruction Activities
In reconstruction activities, students use the original, unmodified text to select specific information and represent it in a different form, thereby developing their analytical skills.
🔧 핵심 정정
위 설명은 analysis activities의 정의입니다. Reconstruction activities는 원본 텍스트를 변형(modified)하여 제공하고, 학생이 예측·재구성하는 활동입니다.
📖 상세 설명
Directed Activities Related to Texts (DARTs): 학생들이 텍스트와 상호작용하며 읽기 이해력을 높이고 critical reader가 되도록 하는 활동 (Nunan).

Reconstruction activities: modified text 사용 → 원본을 분절·공란 처리 → 학생이 예측하며 재구성 → text type 구조 이해 발달. 예: cloze (text completion), sequencing, prediction, table completion.

Analysis activities: unmodified text 사용 → 특정 정보 추출 후 다른 형식으로 표현 → 분석 기능 발달. 예: underlining, segmenting, diagrammatic representation, summary.
Reconstruction activities: The original text is broken down and given to Ss either in segments or as blocks of text with gaps. Ss use prediction and reconstruct. / Analysis activities: Ss select specific information from the text and represent it in a different form using unmodified text.
📊 비교 정리 — Reconstruction vs Analysis
Reconstruction
변형 텍스트 사용 / 예측·재구성 / cloze, sequencing, prediction / text 구조 이해
Analysis
원본 텍스트 사용 / 정보 추출·변환 / underlining, segmenting, diagrammatic rep. / 분석 능력 발달
5
기출 2017.A.6 Skimming / Scanning
Skimming refers to reading carefully for a particular piece of information without necessarily understanding the rest of the text, while scanning is reading quickly to gain a general idea of a passage.
🔧 핵심 정정
설명이 서로 뒤바뀌었습니다. Skimming은 전체적인 주제·요지 파악을 위해 빠르게 읽는 것이고, Scanning은 특정 정보를 찾기 위해 나머지는 이해하지 않아도 되는 읽기입니다.
📖 상세 설명
Skimming: Glancing rapidly through the text to get a general idea. 학생들이 약 30초 동안 텍스트를 훑으며 주제·요지를 파악. 매 단어에 집중하지 않음.

Scanning: Glancing rapidly through the text to locate specific information. 이미 학습한 discourse markers 등을 빠르게 찾을 때 사용.
Skimming: Glancing rapidly through the text to get a general idea. / Scanning: Glancing rapidly through the text to locate specific information.
📊 비교 정리 — Skimming vs Scanning
Skimming
목적: 전체 주제·요지 파악 / 빠르게 훑기 / ~30초 / 매 단어 집중 불필요
Scanning
목적: 특정 정보 찾기 / 나머지 이해 불필요 / 특정 단어·숫자·discourse markers 탐색
6
기출 2009.25 · 2014.10 Think-aloud / Metacognitive
The think-aloud protocol is a technique in which the teacher encourages students to monitor their comprehension process while reading and report on what they are doing, thereby fostering metacognitive awareness.
✅ 핵심 확인
정확한 설명입니다. Think-aloud protocol은 읽는 과정에서 자신의 사고 과정을 소리 내어 말하게 함으로써 metacognitive awareness를 기르는 핵심 전략 지도 기법입니다.
📖 상세 설명
Think-aloud protocol: T encourages Ss to monitor their comprehension process while reading. They report on what they are doing and their thoughts → Ss can verify strategies that assist them in getting at the meaning. inferencing 과정을 파악하는 데도 활용 (2014.10 기출).

Metacognitive awareness: 학습자가 자신의 정신적 처리과정을 인식하는 학습 과정. Before reading → 전략 선택, While reading → 전략 모니터링, After reading → 전략 효과 평가.
Think-aloud protocol: T encourages Ss to monitor their comprehension process while reading. At the same time, they are required to report on what they are doing and their thought to the teacher. → metacognitive awareness (기출 2009.25, 2014.10 공통 핵심 키워드)
7
기출 2009.15 Graphic Organizer
A graphic organizer is an analysis activity that uses unmodified text to help students visually represent relationships among ideas, and it is associated with top-down processing.
✅ 핵심 확인
정확한 설명입니다. Graphic organizer는 분석 활동(analysis activity)으로서 원본 텍스트를 사용하며, top-down processing과 연결됩니다 (기출 2009.15).
📖 상세 설명
Graphic organizer: A visual representation to show relationship among ideas in a text. 학생들이 텍스트의 핵심 정보, 정보 조직 방식, 아이디어 간 관계를 시각적으로 파악. 이 정보를 바탕으로 말하거나 쓰는 활동으로 확장 가능.

기출 2009.15에서는 학생들이 읽은 텍스트를 시각 자료로 표현하는 수업으로 출제. top-down processing 활동으로 분류 (diagrammatic representation의 대표 예시).
Graphic organizer: A visual representation to show relationship among ideas in a text. Students are able to see the key information in a text, the organization of text information, the ways that information is structured, and the relationships among ideas presented in a text.
8
기출 2020.A.10 Outlining / Inferencing
Inferencing is a reading strategy that helps readers organize a text by distinguishing and listing main ideas and supporting details, giving an organized representation of the information.
🔧 핵심 정정
위 설명은 outlining의 정의입니다. Inferencing은 텍스트에 명시되지 않은 아이디어를 파악하기 위해 맥락적 단서를 활용하여 추론하는 전략입니다.
📖 상세 설명
Inferencing: Identify ideas that are not explicitly stated. Look for the clues in the text and use these clues to guess about the text and about the writer's ideas. inferential level of comprehension과 연결. think-aloud data가 inferencing 과정 파악에 활용됨.

Outlining: Distinguish and list the main ideas and supporting details of a text → gives readers an organized representation of the information. 텍스트의 계층 구조를 파악하는 분석 전략.
Inferencing: identify ideas that are not explicitly stated, look for the clues in the text and use these clues to guess about the text and about the writer's ideas. / Outlining: distinguish and list the main ideas and supporting details of a text.
📊 비교 정리 — Inferencing vs Outlining
Inferencing
명시되지 않은 의미 추론 / 맥락 단서 활용 / inferential comprehension level
Outlining
주제문·뒷받침 문장 구분·목록화 / 텍스트 전체 구조 파악 / 조직적 표현
9
Interactive-compensatory Model
The interactive-compensatory model proposes that a deficiency in processing at one level can be compensated by reliance on processing at another level, so a reader who lacks automatic word recognition can rely more on contextual information and higher-level schemata.
✅ 핵심 확인
정확한 설명입니다. Interactive-compensatory model의 핵심은 처리 수준 간 결핍의 보완(compensation) 가능성입니다.
📖 상세 설명
Interactive-compensatory model: Language processing is organized into levels ranging from letter recognition to high-level schemata. A deficiency of processing at one level can be compensated by reliance on processing at another level.

예: fast automatic word recognition 능력이 부족한 학습자 → contextual information·고수준 schemata를 더 적극적으로 활용하여 보완.

Interactive processing(단순)과 구별: interactive processing은 bottom-up과 top-down이 동시에 작동한다는 것이고, interactive-compensatory는 한 수준의 약점을 다른 수준이 보완한다는 점을 강조.
Interactive-compensatory model: Lg processing is organized into levels ranging from letter recognition to high-level schemata, and a deficiency of processing at one level can be compensated by reliance on processing at another level.
10
기출 2021.A.9 Comprehension Levels
The evaluative (critical) comprehension level requires readers to make personal judgments rather than accepting the author's view, while the inferential level focuses on finding implied meaning not explicitly stated in the text.
✅ 핵심 확인
정확한 설명입니다. 읽기 이해 수준 5단계의 정의가 정확히 기술되어 있습니다.
📖 상세 설명
읽기 이해 수준 5단계:
  1. Literal comprehension level: 텍스트에 명시적으로 표현된 정보 이해·기억
  2. Reorganization (reinterpretation) level: 텍스트 일부를 재조직·재해석 / 기초적 추론
  3. Inferential level: 명시되지 않은 함의 파악 / 독자 경험·직관 활용 추론
  4. Evaluative (critical) level: 저자 의도 수용 대신 개인적 판단 / 작가의 편견·가치·사실성 평가
  5. Creative level: 정서적·미적 반응 / 결말 상상 / 독서록 작성
Inferential level: comprehension level to find intention which is not explicitly stated in a passage, using the reader's experience and intuition. / Evaluative level: comprehension level to make personal judgment instead of accepting the author's intention.
📊 5단계 이해 수준 한눈에
Literal → Reorganization → Inferential → Evaluative → Creative

🧠 ① Background Knowledge in Listening Schematic Knowledge ▼
Core Principle
The lack of shared schematic information makes comprehension difficult or even impossible.
Paused Listening Tasks 기출 포인트
Paused listening tasks: An effective technique that shows the presence of relevant schematic knowledge facilitates comprehension.

T divides recordings into segments → plays at regular intervals of pauses

During each pause: Ss write the topic they believe + words/phrases that made them think so

Pausing = a kind of scaffolding → enhances metacognitive awareness

Clues activate competing content schemata

→ Process-oriented listening: develops effective self-regulating skills

Content schemata
world / topic knowledgesociocultural knowledge
Formal schemata
rhetorical / discourse structure글의 형식에 대한 사전 지식
📥 ② Input Roughly / Finely-tuned · Authentic ▼
Roughly-tuned vs Finely-tuned Input
Roughly-tuned input
학습자 숙달도에 대략적으로 맞춤 ≈ natural usage of language authentic material · focus on meaning
Finely-tuned input
학습자가 배운 구조·어휘만 포함 carefully controlled graded readers · target grammar 예시
Concocted Texts vs Authentic Texts 기출 포인트
Concocted texts: give false expectation about real-life listening experiences → transfer 방해

Unnaturally lower speed → damages text authenticity

✅ Better: pauses between phrases and sentences

Authentic texts: expose Ss to naturalness of lg · real-life listening experience

🎯 ③ Strategies for Authentic Materials 3 Strategies ▼
Simplifying the tasks
언어(lg) 유지 task를 단순화 self-confidence↑ · motivation↑
Grading the texts
텍스트 자체를 난이도 순으로 배열 lower-level → advanced
Staging the listening
활동을 점진적으로 구성 simple → demanding
🔑 핵심 구분

Simplifying = task 단순화 (≠ language 단순화)

Grading = 텍스트 수준별 배열

Staging = 활동 점진적 구성 (같은 텍스트 가능)

🗣️ ④ Functions of Listening Transactional · Interactional ▼
Transactional function
message-oriented propositional / factual information 전달 confirmation checks 사용 giving & obtaining goods/services
Interactional function
person-oriented social relationship 형성·유지 phatic communion e.g. "Nice day, isn't it?"
⚙️ ⑤ Comprehension Processing ★★★ 최빈출 ▼
Bottom-up Processing
Listening is a process of decoding the sounds that one hears in a linear fashion. Moving from the parts to the whole.

Focus on linguistic form

Exercises: a) structural system of English 다루기 b) listening for details — discrete items 선택

Top-down Processing
Ls actively reconstruct the original meaning using incoming sounds as clues + prior knowledge of context and situation.

Focus on meaning · global listening strategies

Relies on extralinguistic skills

Contextual clue + Schemata = Reconstruct meaning

Interactive Processing 기출 포인트
Top-down and bottom-up processing are simultaneously used. Caters to individual learning styles.

Level-differentiated lesson: beginners → bottom-up focus

Level-differentiated lesson: advanced Ls → top-down focus

Predicting(pre) · Monitoring(while) · Evaluating(post) = metacognitive

Inferencing ≠ Scanning: Inferencing = between the lines / Scanning = pick out specific info

① Pre-listening ★★ 기출 포인트

Pre-teaching critical vocabulary: facilitates comprehension · do NOT pre-teach ALL unknown words · teach lexical chunks too

Establishing context: provide contextual info related to excerpt

Creating motivation:

Signposting Qs
T-provided · guide direction
Generating Qs
S-created · title 제공 → S가 질문 생성

Activating schemata: brainstorming / visuals

② While-listening

Extensive phase (1st): general idea 파악 · confirm predictions

Intensive phase (2nd+): T provides pre-set Qs → Ss find specific info · submit answers

Comprehension checking: Doing · Choosing/matching · Transferring · Transcribing · Answering · Condensing · Extending

③ Post-listening 기출 포인트

Skill integration: integrate other skills (speaking/writing/reading) with listening

Functional lg: T draws attention to functional uses (refusing, apologizing...)

Inferring unfamiliar vocabulary: controlled practice

Reflection: reflect on learning · plan for next listening

Checking comprehension / summarizing / discussion

Product vs Process Oriented Listening
Product-oriented
'Listen to learn' use what Ss comprehend → complete another activity write a report · give a speech
Process-oriented
'Learn to listen' focus on how understanding takes place metacognitive awareness↑ · self-regulation
✏️ ⑨ Listening Activities Dictation · Dictogloss · Info Transfer ▼
Dictation
Dictation: an imitative writing activity — T reads aloud in short phrase units with pauses → Ss write exactly what they heard.

Purpose: generate an exact/accurate replica

Unnatural · no regard in meaning

Variation: Cloze type of dictation

Dictogloss 기출 포인트
Dictogloss: A collaborative outcome task where Ls work together to reconstruct a text presented orally.
Preparatory
→
Dictation
meaning + details + note-taking
→
Reconstruction
→
Analysis & Correction

Ss utilize both bottom-up AND top-down strategies → interactive processing

Jigsaw Listening
Based on information-gap principle: different groups listen to different recordings → share information to complete the whole picture.

Cooperative learning 촉진

Information Transfer
Information Transfer: reinterpret and transform information from one mode to another (aural → visual representation).

Visual support → deep processing · more mental effort

Designed for scanning / selective listening → integrative activity

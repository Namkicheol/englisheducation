🔄 ② Interlanguage ★★★ 최빈출 ▼
특징
Interlanguage: An intermediate system between L1 and TL — systematic, variable, dynamic. Own unique, coherent rule system.
Developmental Stages ★★★
Stage특징Self-correction
Presystematicrandom errors · vague rule awareness❌
Emergentsome consistency · backsliding · uptake ❌❌ (with feedback)
Systematicconsistent · uptake ✅✅ (with feedback)
Stabilizationfew errors · fossilization 가능✅ (without feedback)

Backsliding: 규칙을 이해한 것 같지만 이전 단계로 퇴행하는 현상 (emergent stage)

Restructuring: interlanguage 발달 과정에서의 질적 변화

Fossilization 기출 포인트
Fossilization: relatively permanent incorporation of incorrect linguistic forms into L2 competence.

원인: ① lack of negative feedback ② insensitivity to feedback ③ positive feedback이 오류 강화 ④ too much automatization

예방: explicit corrective feedback으로 오류 인식 유도

💬 ④ Corrective Feedback Types ★★★ 최빈출 ▼
TypeImplicit/Explicit특징예시
Recast Implicit erroneous utterance 재구성 · flow 유지 · positive+negative evidence · 학습자가 confirmation으로 오해 가능 S: "I goed." T: "Oh, you went?"
Clarification request Implicit self-repair 유도 · communication breakdown 가능 · 가장 효과적 "I'm sorry?" / "Pardon?"
Metalinguistic feedback Implicit/Explicit 오류 성격 설명 · correct form 제공 X "What is the past tense?"
Elicitation Implicit/Explicit incomplete sentence/question → self-correct 유도 "He comes... / We are...?"
Explicit correction Explicit 오류 직접 명시 + correct form 제공 "No, not 'goed', say 'went'."
Repetition Implicit 오류 부분만 반복 · intonation 변화 S: "I goed." T: "Goed?"
⚠️ Recast 핵심 정리 (★★★ 최빈출)

① Implicit · communicative flow 유지

② Positive evidence(correct form) + Negative evidence(오류 암시) 동시 포함

③ 학습자가 confirmation check로 오해 가능 → unsuccessful uptake

④ Meaning에 집중하느라 form 수정 인식 못할 수 있음

⑤ Intonational signal 추가 → explicitness↑

🔁 ⑤ Uptake · Modified Output · Pushed Output ★★★ 최빈출 ▼
개념정의핵심
Uptake T's corrective feedback에 즉각적으로 뒤따르는 학습자 발화 successful / unsuccessful / no uptake / acknowledgement
Modified output 피드백 이후 실제로 수정이 이루어졌는지 여부 수정 여부 focus · not all uptake = modified
Pushed output T가 학습자를 올바른 형태로 push하는 과정 not all pushed = modified output
🔑 3개념 구분 공식

Uptake = feedback 직후 즉각 반응 (수정 여부 무관)

Modified output = 수정 결과에 focus (successful / unsuccessful)

Pushed output = T가 push하는 과정 → 반드시 modified는 아님

❓ ⑦ Display Q · Referential Q · IRE · IRF ★★★ 최빈출 ▼
Display Q vs Referential Q ★★★
Display question
T already knows the answer 학습자 이해도 확인 목적 e.g. "What is the past tense of go?"
Referential question
T does NOT know the answer authentic communication 촉진 e.g. "What did you do last weekend?"
IRE vs IRF 기출 포인트
IRE (Initiation–Response–Evaluation)
T initiates → S responds → T evaluates display question 중심 teacher-centered · form check
IRF (Initiation–Response–Feedback)
T initiates → S responds → T gives feedback referential question 가능 더 authentic한 interaction
